# #91 Speak Time Snap

A high-precision script duration calculator for video creators, presenters, and narrators. Instantly estimate reading time with smart language detection and real-time "Goal Time" feedback.

## Core Philosophy
- **1 Tool / 1 Action**: Focuses entirely on converting text to speech duration with zero friction.
- **Privacy First**: No login, no tracking, no ads. Browser-only execution.
- **Fast & Clean**: Optimized for fast-paced professional workflows on both mobile and desktop.

## Key Features
- **Smart Language Detection**: Automatically switches between `Words (Western)` and `Chars (Asian)` based on the script structure.
- **Three-Speed Simulation**: Simultaneously calculates durations for `Slow`, `Normal`, and `Fast` speaking paces.
- **Real-Time Goal Tracker**: Set a "Goal Time" (in seconds) to instantly see how many words/chars you need to add or remove to hit your mark.
- **Pro-Stats Copy**: Tapping the result box copies a professional report (e.g., "150 words / Normal Speed / 01:04") for quick sharing with directors or clients.
- **Mobile Optimized**: Featuring a distraction-free UI with easy "Clear" buttons and iOS-friendly typography.

## Design System
- **Background**: #fff
- **Main Color**: #333
- **Border Radius**: 15px (Enforced across all UI components)
- **Status Indicator**: Dynamic language and count labels for instant validation.

## How to Use
1. Paste your script into the main text area.
2. Check the **Three-Speed** cards to compare different speaking paces.
3. (Optional) Enter a **Goal Time** in seconds to calculate the exact word/char count difference.
4. Tap the **Result Box** to copy the detailed duration report to your clipboard.

## Privacy Policy
- No data collection.
- No cookies.
- No external tracking.
- All script analysis is performed locally in your browser.
